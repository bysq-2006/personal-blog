## 安装环境
首先，在安卓手机上下载并安装 [Termux](https://pan.baidu.com/s/1SMwEFhivbsiGvKmPze5BiQ?pwd=0514) 应用 提取码: 0514

1. 打开 Termux 输入 “pkg upgrade openssl” 这个过程中会提示你是否要更新，输入 y 回车即可。

2. 安装 nginx，输入命令 “pkg install nginx” 回车，等待安装完成。

3. 启动 nginx，输入命令 “nginx” 回车。

4. 打开手机浏览器，输入地址 “[http://127.0.0.1:8080](http://127.0.0.1:8080)”，如果看到欢迎页面，说明 nginx 启动成功。

## 配置网站
1. 进入 nginx 的默认网站目录，输入命令 “cd /data/data/com.termux/files/usr/share/nginx/html” 回车。

2. 我们从手机外部获取制作好的网站（也可以在内部自己制作，通过vim等工具），将网站文件上传到这个目录下。所以我们先获取手机文件访问权限，输入命令 “termux-setup-storage” 回车，允许访问。
<span style="color:#888">~/storage 是 termux-setup-storage 创建的符号链接目录，它指向手机的内部存储（internal storage），相当于 /storage/emulated/0（这是安卓系统的标准路径，表示手机的“根目录”或主存储空间）。</span>

3. 现在，我们需要将网站文件放到 nginx 目录中。为了演示，我准备了一个简单的纯 HTML 示例网站（设计感强，包含渐变背景、卡片布局等现代元素）。你可以从电脑上复制这个示例网站的文件到手机存储，然后在 Termux 中复制到 nginx 目录。

   - 下载示例网站文件：[示例网站下载链接](https://pan.baidu.com/s/176MSKF7rgf5EPU3HM9q7Sw?pwd=0514) 提取码: 0514
   - 将 `index.html` 文件复制到手机的内部存储（这里是 Download 文件夹）。
   - 然后在 Termux 中运行：
     ```
     cp ~/storage/shared/Download/index.html /data/data/com.termux/files/usr/share/nginx/html/
     ```
     （如果文件在其他位置，调整路径）。

4. 重新加载 nginx 配置：`nginx -s reload`。

5. 再次访问 `http://127.0.0.1:8080`，现在应该看到你的自定义网站了！

## 内网传透

如果你想让其他设备访问你的手机网站，可以使用内网穿透工具，比如 [花生壳](https://hsk.oray.com/)。

1. 下载并安装花生壳应用，注册并登录账号。(要下载两个应用，一个是花生壳管理程序，一个是内网穿透客户端)

2. 注册账号并实名认证，然后创建一个内网穿透映射，选择 HTTP 映射（这里要花6块钱）

3. 配置映射，在Termux中查看你的本地IP地址，输入命令 `ifconfig`，找到 wlan0 下的 inet 地址（比如192.168.1.103）

4. 在花生壳的映射配置中，填写本地IP地址和端口8080，保存配置。

5. 启动内网穿透服务，等待映射成功。
